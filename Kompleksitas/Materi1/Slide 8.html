Contoh: Linear Search vs Binary Search

O(n)

Linear Search

  • Memeriksa setiap elemen data satu per satu dari awal hingga akhir.
  • Sederhana, tidak butuh data terurut.
  • Lambat: Semakin besar data, semakin lama waktu yang dibutuhkan.
O(log n)

Binary Search

  • Membagi rentang data menjadi dua bagian di setiap langkah.
  • Sangat Cepat: Efisien bahkan untuk jutaan data.
  • Syarat Wajib: Data harus sudah dalam kondisi terurut (sorted).

Kesimpulan: Pemilihan algoritma yang tepat dapat menghemat waktu eksekusi secara drastis, terutama pada skala data besar.
